## Decisions

- CI runs on Ubuntu deliberately. Everything so far has been built and tested on one macOS
  machine, and the probes lean on shell tools whose flags differ between platforms. CI is the
  cheapest way to stop that gap growing.
- No `npm install` step: harness-kit has zero runtime dependencies, and keeping it that way is
  a constitution-level decision.
